# Code Base CSE3000
## Libraries Used

- **Dear ImGui** (https://github.com/ocornut/imgui)
- **GLFW** (https://github.com/glfw/glfw)
- **OpenGL** 

## Prerequisites
- CMake 3.31 or higher
- C++ compiler supporting C++20
- OpenGL

## How to Run

1. Open the project in a CMake-compatible IDE.
2. Configure CMake.
3. Build the project.
4. Run the application.
